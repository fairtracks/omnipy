# Install

Install Omnipy in a Python environment and verify it is available.

## Requirements

- Python **3.10–3.13**

## Quick start install (`pip`)

For a fast path to usage:

```bash
python -m pip install omnipy
```

## Development-style setup (`uv`)

If you already use `uv` and virtual environments:

```bash
uv venv
source .venv/bin/activate
uv pip install omnipy
```

## Verify

```bash
python -c "import omnipy; print(omnipy.__version__)"
```
