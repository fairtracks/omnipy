# Install

Install Omnipy in a Python environment and verify it is available.

## Requirements

- Python **3.10–3.13**

## Install with `uv`

```bash
uv venv
source .venv/bin/activate
uv pip install omnipy
```

## Verify

```bash
python -c "import omnipy; print(omnipy.__version__)"
```
